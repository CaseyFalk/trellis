This is a **Trellis** learning project. Open it with your AI coding agent, ask
questions about your focus area, and run capture when understanding settles — pages
file themselves into the right section and cross-link automatically.

This page, the section intros, and the tag vocabulary are placeholders until you
run first-session tailoring. Your agent will offer to do that the first time you
open the project.
