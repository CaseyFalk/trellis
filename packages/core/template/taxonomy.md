# Tag Vocabulary

The controlled list of tags pages may use. Tags come from this list; coining a new
one is a deliberate act surfaced in the capture plan, then added here — this keeps
the search tag-filter clean (no `perf`/`performance` duplicates).

Format: each tag is a list item — a dash, the tag wrapped in backticks, an em-dash,
and a short gloss describing what it groups. (`trellis check` reads exactly that
shape, so keep examples out of the list.)
